# AppInfo

AppInfo — это приложение для Android, предназначенное для просмотра информации об установленных приложениях на устройстве. Оно позволяет получить данные о названии, версии, имени пакета и контрольной сумме (SHA-256) APK-файла, а так же запускать необходимое приложение.

## Особенности

- **Список установленных приложений**: Просмотр всех приложений, установленных на устройстве.
- **Детальная информация**: Просмотр имени, версии, имени пакета и иконки приложения.
- **Контрольная сумма SHA-256**: Вычисление и отображение SHA-256 хеша APK-файла при открытии деталей.
- **Асинхронная загрузка**: Информация и иконки загружаются асинхронно, что обеспечивает плавность интерфейса.
- **Индикаторы загрузки**: Визуальные индикаторы показывают, когда данные загружаются или вычисляются.
- **Современный UI**: Интерфейс построен с использованием Jetpack Compose.

## Технологии

- **Язык программирования**: Kotlin
- **Фреймворк UI**: Jetpack Compose
- **Архитектура**: MVVM (Model-View-ViewModel)
- **Асинхронность**: Kotlin Coroutines и Flow
- **Навигация**: Navigation Compose
- **Отображение изображений**: Coil-Compose
- **Криптография**: `java.security.MessageDigest` (SHA-256)
